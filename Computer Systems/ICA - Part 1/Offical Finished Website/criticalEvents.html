CRITICAL EVENTS

One of the large issues which is faced is how to handle “critical events” these are described as actions which are taken in a crisis.

Deciding how the cars will handle these is incredibly complicated as there are not only the legal issues of blame involved, but the machine will need to be able to weigh different outcomes financially and morally.

The financial implications are a lot simpler, if the event is going to cause financial damage then the lowest value will always be chosen. The moral problems are a lot more difficult when deciding how to take action.

Researchers at MIT have created a site which explores this issues by giving you the choice between scenarios E.G The car will hit a small child, or an elderly woman, how would you choose?
